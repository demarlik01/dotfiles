# Neovim 단축키 모음

## Leader 키

`Space`를 leader 키로 사용한다.

## Neo-tree (파일 트리)

### 열기/닫기

| 키 | 동작 |
|---|------|
| `Space + e` | 파일 트리 토글 (열기/닫기) |
| `Space + o` | 파일 트리로 포커스 이동 |

### 트리 안에서

| 키 | 동작 |
|---|------|
| `Enter` | 파일 열기 / 폴더 열기 |
| `s` | 세로 분할로 열기 |
| `S` | 가로 분할로 열기 |
| `a` | 새 파일/폴더 생성 |
| `d` | 삭제 |
| `r` | 이름 변경 |
| `y` | 복사 |
| `x` | 잘라내기 |
| `p` | 붙여넣기 |
| `H` | 숨김파일(dotfiles) 표시 토글 |
| `/` | 파일명 검색 (fuzzy filter) |
| `#` | fuzzy sorter |
| `f` | 필터 (정확한 매칭) |
| `?` | 전체 키바인드 도움말 |

## Telescope (검색)

### 검색 열기

| 키 | 동작 |
|---|------|
| `Space + ff` | 파일 검색 (이름으로) |
| `Space + fg` | 텍스트 검색 (파일 내용에서 grep) |
| `Space + fb` | 열린 버퍼 목록 |
| `Space + fr` | 최근 파일 |
| `Space + fh` | 도움말 검색 |

### 검색창 안에서

| 키 | 동작 |
|---|------|
| `Ctrl+n` / `Ctrl+p` | 아래/위 이동 |
| `Enter` | 선택 |
| `Ctrl+x` | 가로 분할로 열기 |
| `Ctrl+v` | 세로 분할로 열기 |
| `Esc` | 닫기 |

## LSP (코드 분석)

### 탐색

| 키 | 동작 |
|---|------|
| `gd` | 정의로 이동 (Go to Definition) |
| `gr` | 참조 목록 (References) |
| `K` | 문서 팝업 (Hover) |

### 편집

| 키 | 동작 |
|---|------|
| `Space + ca` | 코드 액션 (자동 수정 제안) |
| `Space + rn` | 이름 변경 (Rename) |

### 진단 (에러/경고)

| 키 | 동작 |
|---|------|
| `[d` | 이전 에러로 이동 |
| `]d` | 다음 에러로 이동 |
| `Space + d` | 현재 줄 에러 상세 보기 |

## Vim 기본

### 이동

| 키 | 동작 |
|---|------|
| `h/j/k/l` | 좌/하/상/우 |
| `gg` | 파일 맨 위 |
| `G` | 파일 맨 아래 |
| `Ctrl+d` | 반 페이지 아래 |
| `Ctrl+u` | 반 페이지 위 |
| `w` | 다음 단어 |
| `b` | 이전 단어 |
| `0` | 줄 맨 앞 |
| `$` | 줄 맨 끝 |
| `f{char}` | 현재 줄에서 {char}로 이동 (`;` 반복, `,` 역방향) |

### 편집

| 키 | 동작 |
|---|------|
| `i` | 커서 앞에서 입력 시작 |
| `a` | 커서 뒤에서 입력 시작 |
| `o` | 아래에 새 줄 추가 후 입력 |
| `O` | 위에 새 줄 추가 후 입력 |
| `dd` | 줄 삭제 |
| `yy` | 줄 복사 |
| `p` | 붙여넣기 |
| `u` | undo |
| `Ctrl+r` | redo |
| `ciw` | 단어 지우고 입력 모드 |
| `ci"` | 따옴표 안 내용 지우고 입력 모드 |

### 검색

| 키 | 동작 |
|---|------|
| `/` | 검색 |
| `n` | 다음 검색 결과 |
| `N` | 이전 검색 결과 |
| `*` | 현재 단어 검색 |

### 창 관리

| 키 | 동작 |
|---|------|
| `:vs` | 세로 분할 |
| `:sp` | 가로 분할 |
| `Ctrl+w h/j/k/l` | 창 이동 |
| `:q` | 창 닫기 |
